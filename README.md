# CodeAlpha_NetworkSniffer

This project was developed as part of my Cyber Security Internship with CodeAlpha.

## Description
A Python-based network sniffer built using Scapy to capture and analyze live network traffic.

## Features
- Captures packets in real time
- Displays source and destination IP addresses
- Identifies TCP and UDP protocols
- Shows packet summaries

## Sample Output
The program successfully captured DNS and HTTPS traffic, including requests to location.services.mozilla.com and secure web connections.

## Tools Used
- Python
- Scapy
- Kali Linux
